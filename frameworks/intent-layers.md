# 意圖三層

語意 / 語旨 / 語效。Kim 黃晨皓用來分析溝通本質的框架，特別適用於人際衝突、誤解、說了但對方沒聽進去的情況。

## 三個層次

### 語意（Semantic）

字面上說了什麼？

這是最表面的層次。語意是文字本身的意思，是「我聽到你說的話」。

範例：「你怎麼每次都這樣？」
語意 = 對對方行為的觀察陳述

### 語旨（Illocutionary Force）

說這句話真正的動機和意圖是什麼？

語旨是說話者心裡「真正想要達到的效果」。通常跟語意不同，有時甚至相反。

範例：「你怎麼每次都這樣？」
語旨 = 可能是「我需要被尊重」，或「我很挫折，希望你理解我的感受」，或「我希望你改變某個具體行為」

**關鍵洞察**：大多數溝通問題，根源在於說話者的語旨沒有被對方正確理解。

### 語效（Perlocutionary Effect）

這句話說出去之後，對方實際感受到什麼？

語效是接收者真實的反應和感受。說話者無法控制語效，但可以設計它。

範例：「你怎麼每次都這樣？」
語效 = 對方感到被指責、被否定，立刻進入防禦模式

**問題所在**：語旨想要「被理解」，語效卻是「讓對方防禦」。語旨和語效的落差，就是溝通失敗的根源。

## 溝通分析 SOP

當溝通出現問題時，用三層分析診斷根源：

1. **釐清語意**：「對方字面上說了什麼？」
2. **推測語旨**：「他說這話的真實意圖是什麼？他真正想要的是什麼？」
3. **評估語效**：「我聽到這話之後，我的感受是什麼？這個感受是他想要的嗎？」
4. **找落差**：「語旨和語效的落差在哪裡？是因為表達方式問題，還是資訊不對稱？」
5. **設計新表達**：「如何重新表達，才能讓語效接近語旨？」

## 常見診斷模式

| 衝突類型 | 根源 | 處方 |
|---------|------|------|
| 說了沒人懂 | 語旨 → 語效落差（表達失誤） | 換一種說法，用對方的語言說 |
| 說了對方生氣 | 語效跟語旨相反 | 先處理情緒，再確認語旨 |
| 說得對但沒人聽 | 語旨對但引力不足 | 補充「這對你意味著什麼」 |
| 雙方都覺得對方不理解自己 | 各自的語旨都沒有被語效傳達 | 直接說出語旨：「我真正想說的是…」 |

## 消極句轉問句

三層分析後，把指責性的觀察轉成行動導向的問題：

| 消極句 | 轉換為問句 |
|-------|---------|
| 「他就是不理解我」 | 「我能如何確認他的語旨是否跟我理解的一致？」 |
| 「我說的話他根本沒聽」 | 「我的語效如何？有沒有達到我的語旨？」 |
| 「他的問題是…」 | 「他說這話的語旨是什麼？我有沒有誤讀？」 |

## 觸發問題

- 他說這句話，真正的意圖（語旨）是什麼？
- 我說的話，對方實際感受到什麼（語效）？
- 語旨和語效的落差在哪裡？是表達問題還是理解問題？

# Kim 黃晨皓知識庫

Kim 黃晨皓（阿金）是台灣知名創業教練、思維框架實踐者。本知識庫收錄其核心思維模型、個人特質描述，以及學員視角的第一手觀察。

任何 AI 工具都可以直接讀取本庫的原始 Markdown 文件，無需訂閱或登入。

```

├── llms.txt                  # AI 入口索引（建議從這裡開始）

│   ├── building-blocks.md    # 積木思維

├── profile/                  # Kim 是誰

│   └── fish-reflection.md    # 學員心得
└── transcripts/              # Phase 2：課程影片逐字稿（待更新）
```

## 如何用 AI 讀取本知識庫

**方法一：貼入 llms.txt 網址（推薦）**

將以下網址貼入任何支援讀取 URL 的 AI（ChatGPT、Claude、Gemini 等）：

```
https://raw.githubusercontent.com/fishtvlvoe/kim-knowledge/main/llms.txt
```

AI 會自動識別索引，按需讀取各文件。

**方法二：直接貼入單一文件的原始網址**

例如讀取積木思維框架：

```
https://raw.githubusercontent.com/fishtvlvoe/kim-knowledge/main/frameworks/building-blocks.md
```

**方法三：複製文件全文**

在 GitHub 上打開任一 `.md` 文件，點選「Raw」按鈕，複製全文貼入 AI。

## 如何貢獻

1. Fork 本 repo
2. 在對應目錄建立或修改 `.md` 文件
3. 遵循格式規範：H1 標題、純文字（無 HTML、無圖片）
4. 送出 Pull Request，說明修改原因

Google Drive 課程資料會透過 GitHub Actions 定期自動同步（每日台灣時間 10:00）。
